# Changelog

All notable changes to this project will be documented in this file.

---

## [v0.6] - 2026-07-13

### Added
- Added calculation validation to prevent displaying undefined results.
- Added a retry loop for invalid operations.

### Changed
- Improved variable naming for readability.

---

## [v0.5] - 2026-07-13

### Added
- Added validation for numerical input using `try`/`except`.
- Added input loops that repeatedly prompt the user until valid numbers are entered.

---

## [v0.4] - 2026-07-13

### Added
- Added protection against division by zero.

---

## [v0.3] - 2026-07-13

### Added
- Added validation for invalid mathematical operations.
- Added an error message for unsupported operators.

---

## [v0.2] - 2026-07-12

### Added
- Initial calculator implementation.
- Support for addition (`+`), subtraction (`-`), multiplication (`*`), and division (`/`).
- Support for decimal calculations using floating-point numbers.

## [v0.1] - 2026-07-13

### Added
- Initial project created.
- Supported addition (`+`) only.

> **Note:** Version `v0.1` predates this repository's Git version history and therefore does not have a corresponding Git tag.
