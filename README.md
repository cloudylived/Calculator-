# My First Python Project

## Calculator

A simple calculator built in Python to practise programming fundamentals and learn core Python concepts.

## Current Version

Version 0.7 - Multiple Calculation Support

## Features

- Performs basic arithmetic operations:
  - Addition (+)
  - Subtraction (-)
  - Multiplication (*)
  - Division (/)
  - Exponentiation (**)
- Supports decimal calculations using floating-point numbers
- Validates user input for numbers and operations
- Prevents division by zero
- Allows users to perform multiple calculations without restarting the program
- Validates user choices when continuing or exiting the calculator

## How to Run

1. Make sure Python is installed
2. Download or clone this repository
3. Run the calculator:

```bash
python calculator.py
```
